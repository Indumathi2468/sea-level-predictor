# Sea Level Predictor 

This project analyzes global average sea level change data from 1880 and predicts sea level rise through 2050 using linear regression.

## Tools Used
- Python
- Pandas
- Matplotlib
- SciPy

## Project Tasks
- Imported dataset using Pandas  
- Created scatter plot of sea level vs year  
- Applied linear regression for:
  - All data (1880–latest)
  - Recent data (2000–latest)
- Predicted sea level rise for 2050
